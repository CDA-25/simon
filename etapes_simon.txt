Étapes pour créer le jeu Simon :

1. Préparer la structure HTML :
   - Créer une zone de message pour afficher les instructions et les résultats.
   - Ajouter 4 blocs de couleurs (vert, rouge, jaune, bleu) avec des IDs distincts.
   - Ajouter un bouton "Start Game" pour lancer la partie.
   - Afficher le score du joueur.

2. Styliser le jeu avec CSS :
   - Disposer les 4 blocs de couleur de façon visible et attrayante.
   - Appliquer des couleurs de fond transparentes pour chaque bloc.
   - Styliser le bouton de démarrage et la zone de score.
   - Ajouter un effet visuel pour les blocs désactivés (ex : opacité réduite).

3. Initialiser le script JavaScript :
   - Sélectionner tous les éléments nécessaires (blocs, bouton, message, score).
   - Définir les variables pour la séquence de Simon, la séquence du joueur, le score, etc.

4. Gérer la phase "Simon says" :
   - Générer une séquence de 3 couleurs aléatoires au début.
   - Afficher la séquence en allumant les blocs dans l'ordre (effet visuel clair).
   - Bloquer les clics utilisateur pendant cette phase.

5. Gérer la phase "Player repeat" :
   - Débloquer les clics sur les blocs.
   - Enregistrer la séquence jouée par l'utilisateur.
   - Appliquer un effet visuel différent lors du clic utilisateur.

6. Vérifier la réponse du joueur :
   - Comparer la séquence du joueur à celle de Simon au fur et à mesure.
   - Si la séquence est correcte, augmenter le score et ajouter une nouvelle couleur à la séquence.
   - Si la séquence est incorrecte ou si le joueur ne répond pas en 5 secondes, afficher un message de fin et bloquer le jeu.

7. Boucle de jeu :
   - Répéter les phases tant que le joueur ne se trompe pas.
   - Permettre de relancer une partie avec le bouton "Start Game".

8. Bonus :
   - Ajouter des effets visuels supplémentaires (animations, sons).
   - Afficher le meilleur score ou un classement si besoin.

Résumé :
- 2 phases : "Simon says" (affichage séquence) puis "Player repeat" (réponse utilisateur).
- Effets visuels différents pour Simon et le joueur.
- Blocage des clics pendant la séquence de Simon.
- Timeout de 5 secondes pour la réponse.
- Ajout d'une couleur à chaque réussite.
