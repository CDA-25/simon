# Challenge Simon

Bon on va pas ce mentir pour le moment on a joué à la "dinette" maintenant on va faire un vrai  
jeu de "bonhome" ! Le jeu "Simon says"

Pour ceux qui ne connaissent pas, Simon est un jeu de memory électronique, composé de 4 cases colorées (rouge, bleu, vert, jaune). Le jeu se décompose en 2 phases: 
- "Simon says" : 3 cases s'allument, dans un ordre aléatoire.
- "Player repeat": le joueur (donc nous!) doit reproduire la séquence de Simon.

Si la réponse est juste, Simon ajoute une nouvelle couleur à la fin de la séquence, et recommence.
Si le joueur se trompe ou ne répond rien pendant 5 secondes, la partie s'arrête.

Donc le jeu dois comporter
- Une zone de message (pour afficher les messages d'information à l'utilisateur)  
- Les 4 blocs de couleurs du jeu (les couleurs sont données un peu plus haut)
- Chaque bloc sont cliquable par l'utilisateur afin de stocker la séquence qu'il execute
- Il faut que les blocs donne un effect qu'on comprenne que la couleurs à été jouer par "Simon"  
ou par le "user", utiliser un effet de couleur differente (plus clair, plus foncé comme vous voulez)
- Penser à bloquer les click sur les buttons quand "Simon says" pour pas que l'utilisateur clique en  
meme temps que simon donne sa séquence
